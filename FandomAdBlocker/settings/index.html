Fandom Ad Blocker is a browser extension designed to block ads on Fandom websites. It allows users to customize their ad-blocking experience by adding specific fandoms to an allow list or blocking cookies from certain fandoms. This extension is open-source and 100% free to use.
Latest update changelog here.

For more information, visit the GitHub repository.
Features
Customizable Allow List
Add specific fandoms to an allow list to prevent ads from being blocked on those pages.
Cookie Blocking
Block cookies from specific fandoms to enhance privacy and reduce tracking.
Statistics
View detailed statistics on blocked ads, including the number of ads blocked and the types of elements targeted.
Open Source
The extension is open-source, you can add your suggestions and contributions on the GitHub repository.
Privacy
Data Collection
This extension does not collect any personal data. All data is stored locally on your device. You can view it on statistics page.
Need help? Open a new issue on the Issues page on GitHub repository. I'll try my best to answer any issues as fast as possible.
FAQ
Why are some ads still showing?
The extension is not 100% perfect and some ads may still slip through. If you encounter an ad that is not blocked, please report it on the Issues page on GitHub repository.
Does this extension work with other ad blockers?
Yes, it works perfectly with any other ad blocker.
Does this slow down my browser?
This extension is pretty lightweight compared to most other ad blockers, it's designed to block specific ads on Fandom.com and not everywhere.
So compared to other ad blockers which block "all" ads this one is much more lightweight.
However, it may still have a small impact on performance, especially on pages with a lot of ads.
